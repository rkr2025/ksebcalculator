Instructions for Using the Solar Prosumers Electricity Bill Calculator

  1. Select the appropriate phase (Phase 1 or Phase 3) from the dropdown menu.
  2. Select the Meter Owner (KSEB or Consumer) from the dropdown menu.
  3. Enter the Solar Generation Meter Reading (in kWh) (Current Reading - Initial Reading)in the provided input box.
  4. Enter the KSEB Net Meter Reading for both Import Total (in kWh) and Export Total (in kWh) in the respective input boxes.
  5. Import Total (in kWh) [Current Import Reading - Initial Import Reading]
  6. Export Total (in kWh) [Current Export Reading - Initial Export Reading]
  7. Click the "Bill Calculation" button to calculate your electricity bill.
  8. The results, including your total units consumed and detailed bill information, will be displayed below the form.
  9. If you want to reset all the input fields and clear the results, click the "Reset" button.
  10. There is an option to print the bill details at the end of the page.
